# AceZone

AceZone is a full-stack web application designed for sports equipment shopping and rental services.  
The platform allows users to browse products, manage favorites, rent or purchase equipment, and interact with a modern e-commerce experience.

## Features

- User authentication and authorization
- Sports equipment shopping and rental system
- Product catalog with detailed product pages
- Shocpping art and favorites functionality
- Admin dashboard for product management
- Add, edit, and delete products
- Image upload support for products
- Responsive user interface

## Tech Stack

### Frontend
- React
- JavaScript
- CSS

### Backend
- Java Spring Boot
- REST API

### Database
- SQL Database
